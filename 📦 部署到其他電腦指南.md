# 📦 CWA 階梯式報價系統 - 部署到其他電腦指南

## 🎯 快速摘要

如果您想在其他電腦使用本系統，根據使用情境有不同的檔案需求：

---

## 📋 情境一：只需瀏覽器開啟使用（最簡單）

### 需要的檔案
```
✅ 🐟 CWA 階梯式報價系統（整合版）.html  （必要）
```

### 適用情境
- 商品資料已內建在 HTML 檔案中
- 不需要更新商品資料
- 只要開啟檔案就能使用

### 部署步驟
1. 複製 HTML 檔案到目標電腦
2. 雙擊開啟（會使用預設瀏覽器）
3. 使用「團主」登入，密碼：`admin`

### ⚠️ 注意事項
- 客戶和訂單資料存在瀏覽器的 LocalStorage
- 不同瀏覽器的資料不互通
- 清除瀏覽器快取會刪除所有資料
- 記得定期使用系統內的「備份」功能

---

## 📋 情境二：需要自動更新商品資料（完整功能）

### 需要的檔案結構
```
專案資料夾/
├── 🐟 CWA 階梯式報價系統（整合版）.html  （主系統）
├── auto-update-and-start.py                （自動更新腳本）
├── 啟動系統.command                        （Mac 啟動器）
├── 報價單資料/                            （資料夾）
│   └── 進口商品階梯式報價.xlsx            （Excel 報價單）
└── 備份/                                  （自動建立）
```

### 適用情境
- 需要定期更新商品價格和庫存
- 商品資料來自 Excel 報價單
- 想要自動化更新流程

### 部署步驟

#### Windows 電腦
1. 安裝 Python 3
   - 下載：https://www.python.org/downloads/
   - 安裝時勾選「Add Python to PATH」
2. 安裝必要套件：
   ```batch
   pip install pandas openpyxl
   ```
3. 複製以下檔案：
   - `🐟 CWA 階梯式報價系統（整合版）.html`
   - `auto-update-and-start.py`
   - `報價單資料/` 資料夾（含 Excel 檔案）
4. 建立啟動批次檔 `啟動系統.bat`：
   ```batch
   @echo off
   python auto-update-and-start.py
   pause
   ```
5. 雙擊 `啟動系統.bat` 啟動

#### Mac 電腦
1. Python 通常已預裝，確認版本：
   ```bash
   python3 --version
   ```
2. 安裝必要套件：
   ```bash
   pip3 install pandas openpyxl
   ```
3. 複製以下檔案：
   - `🐟 CWA 階梯式報價系統（整合版）.html`
   - `auto-update-and-start.py`
   - `啟動系統.command`
   - `報價單資料/` 資料夾（含 Excel 檔案）
4. 給啟動腳本執行權限：
   ```bash
   chmod +x 啟動系統.command
   ```
5. 雙擊 `啟動系統.command` 啟動

## 📋 情境三：需要客戶資料（含歷史資料）

### 額外需要的檔案
```
✅ 客戶資料/
   └── 客戶資料_匯入.csv          （客戶清單）
```

### 部署方式
1. 先按照情境一或情境二部署基本系統
2. 開啟系統後，進入「⬆️ 匯入/匯出」頁面
3. 在「客戶管理」區塊，選擇 CSV 檔案匯入
4. 系統會自動載入所有客戶資料

### CSV 格式要求
客戶 CSV 必須包含以下欄位：
- 客戶代號（如 CU2047X）
- 客戶全稱
- 客戶簡稱
- 電話1
- 組別（店家/工作室）
- 取貨方式
- 其他選填欄位...

## 📋 情境四：完整資料遷移（含訂單記錄）

### 使用系統備份功能

#### 步驟 1：在原電腦匯出備份
1. 登入管理員帳號
2. 進入「⬆️ 匯入/匯出」
3. 點擊「完整備份」
4. 下載 JSON 備份檔案（如 `CWA_完整備份_20260430_1234.json`）

#### 步驟 2：在新電腦匯入備份
1. 先部署基本系統（情境一）
2. 開啟系統，登入管理員
3. 進入「⬆️ 匯入/匯出」
4. 選擇「還原備份」
5. 上傳剛才的 JSON 備份檔案
6. 系統會完整還原：
   - ✅ 所有商品（含階梯價格）
   - ✅ 所有客戶資料
   - ✅ 所有訂單記錄
   - ✅ 系統設定

## 🔒 資料安全提醒

### 瀏覽器資料
- 資料存在 LocalStorage（瀏覽器本地儲存）
- 關閉瀏覽器後資料仍保留
- 但清除快取或重設瀏覽器會清空資料

### 建議備份策略
1. **每週備份一次**
   - 使用系統內建的「完整備份」功能
   - 儲存 JSON 檔案到雲端硬碟

2. **重大操作前備份**
   - 大量匯入資料前
   - 刪除或修改資料前
   - 系統升級前

3. **多地備份**
   - 本地電腦
   - 雲端硬碟（Google Drive、Dropbox）
   - USB 隨身碟

## 🆘 常見問題

### Q1: 在新電腦開啟 HTML，為什麼沒有我的訂單資料？
**A:** 訂單資料存在瀏覽器 LocalStorage，需要使用「完整備份」功能匯出再匯入。

### Q2: 啟動腳本無法執行？
**A:** 檢查：
1. Python 是否已安裝（`python3 --version`）
2. 必要套件是否已安裝（`pip list | grep pandas`）
3. 檔案路徑是否正確
4. Mac 系統需要執行權限（`chmod +x 啟動系統.command`）

### Q3: 可以用 iPad 或手機嗎？
**A:** 可以！HTML 檔案支援行動裝置瀏覽器，但自動更新功能需要電腦環境。

### Q4: 多台電腦的資料如何同步？
**A:** 使用定期備份的方式：
1. 主機定期匯出備份到雲端
2. 其他電腦從雲端下載最新備份
3. 匯入還原即可同步
