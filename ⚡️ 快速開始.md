# ⚡️ CWA代購福利站 - 快速開始

## 🎯 5 分鐘上手指南

---

## 第一步：準備資料（2 分鐘）

### 1. 準備報價單
將供應商的 Excel 報價單放入 `報價單資料/` 資料夾

**檔案格式範例：**
```
報價單資料/
├── 仲發冷凍.xlsx
├── 永豐食品.xlsx
└── 鮮魚批發.xlsx
```

**Excel 內容格式：**
| 品名 | 規格 | 單價／台斤 |
|------|------|-----------|
| 鮭魚 | 11~13斤 | 290 |
| 鱈魚 | 3~5斤 | 420 |

> 💡 檔案名稱會自動成為商品分類

---

### 2. 準備客戶資料
將客戶資料 CSV 放入 `客戶資料/` 資料夾

**檔案格式範例：**
```csv
客戶代號,客戶全稱,客戶簡稱,公司地址,聯絡人1,電話1
1,王大明,大明,台北市,王先生,0912345678
2,李小華,小華,新北市,李小姐,0987654321
```

> 💡 只有「客戶全稱」是必填，其他選填

---

## 第二步：自動載入（1 分鐘）

### 方法一：雙擊執行（推薦）
```
直接雙擊「自動載入資料.command」檔案
```

### 方法二：終端機執行
```bash
python3 自動載入資料.py
```

執行後會看到：
```
✅ 已生成商品匯入檔案: 商品資料_自動匯入_YYYYMMDD_HHMM.csv
✅ 已生成客戶匯入檔案: 客戶資料_自動匯入_YYYYMMDD_HHMM.csv
```

---

## 第三步：啟動系統（1 分鐘）

### 方法一：啟動腳本（推薦）
```
雙擊「啟動系統.command」
瀏覽器會自動開啟系統
```

### 方法二：直接開啟
```
雙擊「🐟 CWA代購福利站.html」
```

---

## 第四步：匯入資料（1 分鐘）

1. **進入系統設定**
   - 點擊上方導航列「🛠️ 系統設定」

2. **匯入商品**
   - 找到「📥 匯入商品」區塊
   - 點擊「選擇檔案」
   - 選擇 `商品資料_自動匯入_XXX.csv`
   - 確認預覽無誤後，點擊「確認匯入」

3. **匯入客戶**
   - 找到「👥 匯入客戶」區塊
   - 點擊「選擇檔案」
   - 選擇 `客戶資料_自動匯入_XXX.csv`
   - 確認預覽無誤後，點擊「確認匯入」

4. **完成！**
   - 點擊「📦 商品管理」查看所有商品
   - 點擊「👥 客戶管理」查看所有客戶

---

## 🎉 開始使用

### 管理員操作
1. 點擊「團主」登入（預設密碼：admin）
2. 進入「📦 商品管理」管理商品
3. 進入「📋 訂單管理」查看訂單
4. 使用「代客下單」功能幫客戶下單

### 客戶操作
1. 點擊客戶名稱登入
2. 瀏覽商品
3. 加入購物車
4. 送出訂單

## 💡 常用功能

### 更新商品價格
```
1. 修改 報價單資料/ 中的 Excel 檔案
2. 重新執行「自動載入資料.command」
3. 重新匯入商品 CSV（系統會自動更新價格）
```

### 新增供應商
```
1. 將新供應商的 Excel 報價單放入 報價單資料/
2. 執行「自動載入資料.command」
3. 匯入新生成的 CSV 檔案
```

### 新增客戶
```
方法一：在系統中手動新增
- 進入「👥 客戶管理」
- 點擊「➕ 新增客戶」

方法二：批次匯入
- 更新 客戶資料/ 中的 CSV 檔案
- 重新執行載入和匯入流程
```

### 備份資料
```
1. 進入「🛠️ 系統設定」
2. 找到「💾 備份」區塊
3. 點擊「完整備份」
4. 下載備份檔案並妥善保存
```

### 匯出訂單
```
1. 進入「📋 訂單管理」
2. 點擊「📥 匯出訂單」
3. 選擇日期範圍
4. 下載 Excel 檔案
```

## ⚠️ 重要提醒

### ✅ 資料會保存
- 匯入的商品、客戶、訂單都會保存在瀏覽器中
- 下次開啟系統，資料會自動載入

### ❌ 清除快取 = 刪除資料
- 清除瀏覽器快取會刪除所有資料
- 請務必定期備份

### 📁 不同瀏覽器資料不互通
- Chrome 和 Safari 的資料是分開的
- 建議固定使用同一個瀏覽器

## 🆘 遇到問題？

### 問題：執行「自動載入資料.command」沒反應
**解決方法：**
```bash
# 檢查 Python 是否安裝
python3 --version

# 安裝必要套件
pip3 install pandas openpyxl
```

### 問題：匯入後沒有商品
**檢查項目：**
1. Excel 檔案中是否有「品名」欄位
2. Excel 檔案中是否有價格欄位（單價、售價等）
3. 是否選擇了正確的 CSV 檔案

### 問題：匯入後沒有客戶
**檢查項目：**
1. CSV 檔案中是否有「客戶全稱」欄位
2. 檔案名稱是否包含「客戶資料」
3. 檔案編碼是否為 UTF-8
