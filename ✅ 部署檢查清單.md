## 🚀 部署到線上

選擇一種方式:

### 選項 A: Firebase Hosting

#### ☐ 10. 安裝 Firebase CLI

```bash
npm install -g firebase-tools
```

- [ ] 已安裝 Firebase CLI
- [ ] 執行 `firebase --version` 確認安裝成功

#### ☐ 11. 登入 Firebase

```bash
firebase login
```

- [ ] 瀏覽器開啟登入頁面
- [ ] 選擇 Google 帳號
- [ ] 授權成功

#### ☐ 12. 初始化專案

```bash
cd /path/to/代購系統
firebase init hosting
```

設定選項:
- [ ] 選擇現有專案: `CWA員工福利站`
- [ ] Public directory: `.` (輸入點)
- [ ] Single-page app: `No`
- [ ] 不要覆蓋 index.html: `No`

#### ☐ 13. 部署

```bash
firebase deploy
```

- [ ] 部署成功
- [ ] 取得 Hosting URL
- [ ] 記下網址: `https://____________.web.app`

### 選項 B: Vercel

#### ☐ 14. Vercel 註冊

- [ ] 前往 https://vercel.com

## 🆘 遇到問題?

### 常見錯誤排查

#### Firebase 連接失敗
1. 檢查 `firebase-config.js` 配置
2. 檢查網路連接
3. 檢查 Firestore 是否啟用

#### 資料沒有同步
1. 檢查 Firestore 規則是否發布
2. F12 查看控制台錯誤
3. 確認網路連接穩定

#### 部署失敗
1. 檢查 Firebase CLI 是否最新版
2. 確認已登入正確帳號
3. 檢查專案選擇是否正確

#### 手機無法使用
1. 檢查瀏覽器版本
2. 清除瀏覽器快取
3. 使用 Chrome 瀏覽器測試
