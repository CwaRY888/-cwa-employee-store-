## 🎯 版本選擇

系統提供兩個核心版本:

### 🖥️ 本地版 (推薦個人使用)
- **檔案**: `🐟 CWA代購福利站.html`
- **優點**: 無需設定，開箱即用
- **適合**: 單人使用、本地測試
- **資料**: 儲存在瀏覽器 LocalStorage

### ☁️ Firebase 雲端版 (推薦多人使用)
- **檔案**: `🐟 CWA代購福利站 (Firebase版).html`
- **優點**: 多人即時同步、雲端備份
- **適合**: 團隊使用、線上部署
- **資料**: 儲存在 Firebase Firestore
- **說明**: 請參考 **🔥 Firebase 設定指南.md**

## 🚀 快速開始

### 🌐 線上部署 - GitHub Pages (3 分鐘) ⭐ 推薦

**最簡單的上線方式！**

1. **上傳到 GitHub**
   - 註冊 GitHub 帳號（免費）
   - 上傳整個資料夾

2. **啟用 GitHub Pages**
   - Settings → Pages → 選擇 main 分支
   - 等 1-2 分鐘

3. **取得網址**
   ```
   https://你的帳號.github.io/cwa-employee-store/
   ```
   分享給員工即可！

**詳細步驟**: 📖 **🌐 GitHub Pages 部署指南.md**

---

### 🖥️ 本地版 (1 分鐘)

1. **開啟系統**
   ```
   雙擊「🐟 CWA代購福利站.html」
   ```

2. **首次使用**
   - 系統會自動載入預設資料（44個商品 + 15位員工）
   - 無需任何設定，開箱即用

3. **登入**
   - 選擇任一員工名稱登入
   - 開始購物！

---

### ☁️ Firebase 雲端版 (10 分鐘) - 進階

**需要即時同步功能才需要**

1. **設定 Firebase**
   - 參考 **🔥 Firebase 設定指南.md**
   - 註冊 Firebase、建立專案
   - 填寫 `firebase-config.js`

2. **測試系統**
   - 開啟 `🐟 CWA代購福利站 (Firebase版).html`
   - 確認連接成功

3. **部署上線**
   - 使用 Firebase Hosting 或 Vercel
   - 分享網址給員工

詳細步驟請參考 **✅ 部署檢查清單.md**

## 🔄 更新商品資料

### 方法一：直接匯入（推薦）
1. 進入「📦 商品管理」
2. 點「📥 匯入報價單」
3. 選擇 Excel 報價單檔案
4. 自動匯入

### 方法二：使用報價單資料夾
1. 將報價單放入 `報價單資料/` 資料夾
2. 使用「📥 匯入報價單」功能
3. 選擇檔案匯入

**報價單格式要求：**
- Excel 格式（.xlsx, .xls）
- 必須包含：品名、價格
- 選填：規格、單位

## ⚠️ 重要提醒

### 資料儲存
- 資料存在瀏覽器 LocalStorage
- 不同瀏覽器資料不互通
- 清除瀏覽器快取會刪除資料

### 建議
- ✅ 定期備份資料
- ✅ 固定使用同一瀏覽器
- ✅ 保留原始報價單 Excel 檔案
- ❌ 不要清除瀏覽器資料

---

## 🧪 測試功能

### 清除資料重新測試
1. 按 `F12` 開啟控制台
2. 輸入：`localStorage.clear()`
3. 按 `Enter`
4. 重新整理頁面（`F5`）
5. 系統會自動載入預設資料
